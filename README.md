## üìñ About The Project

Project Horizon is a modern full-stack web application featuring a dynamic landing page and a powerful admin dashboard. The platform allows administrators to manage projects, clients, and contact inquiries through an intuitive interface. The landing page showcases projects and clients with a beautiful, responsive design, while the admin panel provides comprehensive management capabilities including recent activity tracking, CRUD operations, and image uploads via Cloudinary.
